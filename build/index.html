about the website

The site is generated by Soupault, a static website management tool. The free and open source fonts are set on Lithops, Avara, Cooper*, and Fantasque Sans Mono.

The site is hosted by GitHub Pages. Find the source code on my GitHub profile. The content is CC BY-SA and the code is MIT. There is an Atom feed available for the blog portion of the site.

The domain name is the word and intended name of the website, ‘sunrash’ (with no space), and a reference to the late Sun Ra, the musician:

“Every word was once a poem. Every new relation is a new word.”
(Ralph Waldo Emerson, “The Poet”)
