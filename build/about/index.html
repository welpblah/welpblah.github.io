about the website

The website is built with barf, a website generator in ~100 lines of shell. The fonts are set on EB Garamond and Fantasque Sans Mono.

There is a feed available for the weblog portion of the website: Atom; RSS. As the domain suggets, the website is hosted by GitHub Pages. The source code is available on my GitHub profile.
