about the website

The site is built with Soupault, a static website management tool. The fonts are set on Luciole and Andika.

The site is hosted by GitHub, where the source code is available. The content of the site is CC BY-SA and the code of the site is MIT.

The domain name is the word and intended name of the site, ‘sunrash’ (note: no space), and a reference to the late Sun Ra, the musician.
